## 1. Resumo (PT-BR)
Este projeto tem como objetivo a construção de um braço robótico utilizando tecnologia de impressão 3D, baseado no modelo open source da exiii Inc. A partir do design original, está sendo realizada uma adaptação personalizada para viabilizar a integração com uma Interface Cérebro-Máquina (ICM), com uso de sinais eletromiográficos.

A proposta articula conhecimentos de engenharia física, eletrônica, computação e saúde, combinando prototipagem mecânica, integração de hardware, programação de baixo nível e inteligência artificial aplicada.

## 2. Abstract (EN)
This project aims to develop a 3D-printed robotic arm based on the open-source model from exiii Inc. The original design is being customized to enable integration with a Brain-Computer Interface (BCI), using electromyographic signal processing.

The initiative combines physics engineering, embedded hardware, low-level programming, and AI-based autonomous control, targeting assistive healthcare applications in Brazil.

## 3. Contexto e Motivação
No Brasil, ainda há baixa prioridade estratégica para tecnologias de interface cérebro-máquina e para o desenvolvimento de próteses assistivas funcionais de alto desempenho. Este projeto surge como resposta a essa lacuna, com foco na criação de tecnologia nacional e no desenvolvimento de próteses personalizáveis conforme as necessidades de cada usuário.

Além do desenvolvimento tecnológico, a iniciativa busca fortalecer a formação de engenheiros com atuação prática em saúde, promovendo pesquisa aplicada, inovação local e impacto social.

## 4. Objetivos
- Construir um braço robótico funcional a partir de base open source.
- Adaptar o sistema para integração com ICM por eletromiografia.
- Implementar controle autônomo por IA agêntica com visão computacional.
- Consolidar uma base tecnológica nacional em próteses assistivas.
- Formar estudantes com competências interdisciplinares de alta demanda.

## 5. Escopo Técnico
O desenvolvimento contempla as seguintes frentes:

- Impressão 3D estrutural e prototipagem mecânica.
- Impressão e integração de hardware (placas eletrônicas).
- Programação de baixo nível para controle embarcado.
- Processamento de sinais eletromiográficos.
- Sistema agêntico de IA baseado no framework Agno AI.
- Controle autônomo apoiado por visão computacional.

Os equipamentos de eletromiografia utilizados no projeto são desenvolvidos no Instituto de Física da UFRGS, em parceria acadêmica com o Prof. Mauricio Tosin.

## 10. Impacto Esperado
O projeto visa impactar positivamente o sistema de saúde nacional por meio de uma tecnologia assistiva 100% brasileira, com foco em acessibilidade e funcionalidade para quem realmente precisa.

Também se espera impacto acadêmico e profissional na UFRGS e na comunidade, por meio da formação de engenheiros preparados para atuar em fronteiras tecnológicas como hardware biomédico, IA, manufatura aditiva e interfaces cérebro-máquina.
